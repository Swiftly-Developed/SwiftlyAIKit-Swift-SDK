# ``AppleIntelligenceProvider``

Complete API reference for Apple Intelligence integration.

## Overview

The `AppleIntelligenceProvider` gives you access to on-device Apple Intelligence models with privacy-first local processing and no network requirements.

**Key Features:**
- ✅ On-device processing (no network)
- ✅ Privacy-first architecture
- ✅ Streaming support
- ✅ Local model variants
- ✅ Integration with Apple's Foundation Models
- ✅ No API key required

**Context Window**: Varies by model | **Privacy**: Maximum | **Network**: Not required
