# ``ModelProvider``

Model identification and provider types.

## Overview

The `ModelProvider` enum provides type-safe access to all AI models across providers. It groups models by provider and variant, making it easy to switch between models while maintaining type safety.

**Supported Providers:**
- Anthropic Claude (6 models)
- OpenAI GPT (5+ models)
- Google Gemini (5+ models)
- Perplexity Sonar (3+ models)
- Mistral AI (11 models)
- Cohere Command (11 models)
- DeepSeek (2+ models)
- xAI Grok (7 models)
- Apple Intelligence (local models)
